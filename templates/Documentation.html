Our Project Details and Documentation

  • We used a Python Flask-powered RESTful API.
  • We used Python Jupyter Notebook to clean our data and make predictions.
  • We stored data collected in a SQL database file.
  • We used Javascript to render our data into charts and graphs.

    • We decided to do our project examining the effects COVID19 and interest rates have had on property values.


    The following data sets were used in our analysis:

    1. Zillow property values data, was used to measure the monthly change in property values during the evaluation period. We accessed data for Zillow here.
    2. New York Times data collected for Covid19 cases and deaths was used to evaluate the states, counties, and cities where residents were most affected by the virus. We accessed COVID19 data here.
    3. Census Bureau data was standardized per population, with Covid19 data to determine the number of cases/deaths in each stratum. We accessed Census data here.
    4. We also used interest rates in our analysis.

      5. Visualizations and Pages:

      • Home: Links to all pages
      • Table: This page is where you can search city, county and state level data
      • Housing Analytics: This page has several visualizations
      • Predictions: This page is where we display our use of machine learning to make predictions
      • Documentation: Description of our process
      • About Us: Information about us

      Conclusions and Takeaways:

      • During the evaluation period, COVID-19 cases and deaths increased steadily for several months.
      • Property values increased, in certain states, during the COVID-19 pandemic.
      • We made predictions on the property values based on changes in interest rates for the next four years.
